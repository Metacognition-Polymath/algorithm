### 스터디 룰

- 주 1회 이상 알고리즘 문제를 풀고 공유(해당 레포에 커밋 또는 슬랙 노티 설정)
  - 이 레포에 같이 문제풀이를 올리는 경우 : 본인의 닉네임으로 폴더를 만들어서 파일명 규칙을 지켜서 올린다
    - 본인 닉네임 폴더의 하위 폴더이름은 자유입니다
- 매주 수요일 밤 12시까지(목요일 정산)
- 파일명 규칙
  - [사이트약자]-[문제번호]
  - e.g.,
    - 백준 : bj-101
    - 프로그래머스 : pg-101
    - 정올 : jo-101
    - 코딩도장 : cd-101
    - 구름레벨 : gl-101
    - ...
    - 위 예시에 없는 사이트는 직접 사이트 약어를 추가해주세요 😊
